Unary Computing

UW STACS LAB

A Primer

Unary Computing is an emerging neuromorphic computing paradigm that differs from conventional binary computing. The data in binary computing have multiple parallel bits, with each of different significance. On the contrary, the data in unary computing are in the form of unary bitstreams, with each of identical significance.

According to the bitstream coding, unary bitstreams can leverage either rate coding or temporal coding, which are two fundamental forms in neural coding. Rate-coded unary computing is named stochastic computing, while temporal-coded unary computing is named race logic.
